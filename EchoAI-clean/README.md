# Echo AI

### Voice · Vision · Intelligence

Echo AI is a futuristic AI assistant built with SwiftUI.  
It combines intelligent chat, real-time voice interaction, AI image generation, and a premium dark glassmorphic interface.

---

## Features

- Conversational AI Chat
- Real-Time Voice Assistant
- AI Image Generation (Multiple Styles)
- Live News Feed Integration
- Pro Mode & Advanced Settings
- Modern Glassmorphism UI

---

Built with SwiftUI, Combine, and async/await.
